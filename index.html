Introduction

Once upon a time, at the company I work for, we needed a long list by which users could check multiple items in a dropdown menu similar to Bootstrap's. So we searched the web and... found nothing good (sad). Then came an idea, THE idea to create a boostrap drowndown with checkboxes! And hence, the Dropdown-Checkbox was born.

Pre-requis

  • JQuery 1.9+.
  • Bootstrap 2.3.2+ (less only for the moment)
