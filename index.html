Introduction

Once upon a time, at the company I work for, we needed a long list by which users could check multiple items in a dropdown menu similar to Bootstrap's. So we searched the web and... found nothing good (sad). Then came an idea, THE idea to create a boostrap drowndown with checkboxes! And hence, the Dropdown-Checkbox was born.

Pre-requis

  • JQuery 1.9+.
  • Bootstrap 2.3.2+ (less only for the moment)

Installation

It's so easy:

  1. Add JQuery and Bootstrap.
  2. Add this after html body and JQuery:
    <script type"text/javascript" src="/yourpath/dropdown-checkbox.js"></script>
  3. Add this after the <head> section:
    <link rel="stylesheet" type="text/css" href="/yourpath/">

You're done! Good job :)

Usage

In order to create our dropdown-checbox, you just need to write this small piece of code:

$("myselector").dropdownCheckbox()

Options

Name Default Mandatory Description
data [] Yes

Must have these fields:

  • id
  • label
  • isChecked (optional)
autosearch false No Allow to search when user press a key.
