Browser Compatibility

  • IE 9+
  • Chrome 8+
  • Firefox 10+

Note: Didn't test yet on others browsers.

Example

    Requirements

    • JQuery 1.9+.
    • Bootstrap 2.3.2+ (less only for the moment)

    Note: Didn't test below these versions.

    Installation

    It's so easy:

    1. Add JQuery and Bootstrap.
    2. Add this after html body and JQuery:
      <script type"text/javascript" src="/yourpath/dropdown-checkbox.js"></script>
    3. Add this after the <head> section:
      <link rel="stylesheet/less" type="text/css" href="/yourpath/dropdown-checkbox.less">

    You're done! Good job :)

    Usage

    Add this html where you want:

    <div class="dropdown-checkbox">
  <button class="dropdown-checkbox-toggle btn btn-primary" 
  data-toggle="dropdown-checkbox" type="button">Dropdown Checkbox</button>
  <div class="dropdown-checkbox-content">
    <div class="dropdown-checkbox-header">
      <input class="checkbox-all" type="checkbox"><input type="text" placeholder="Search" class="search"/>
    </div>
    <ul class="dropdown-checkbox-menu"></ul>
  </div>
</div>

    In order to create our dropdown-checbox, you just need to write this small piece of javascript code:

    var myData = [{id: 1, label: "Test" }];
    $("myselector").dropdownCheckbox({ data: myData});

    Options

    Parameter Type Default Required Description
    data Array [] Yes

    Must have these fields:

    • id
    • label
    • isChecked (optional)
    autosearch Boolean false No Allow to search in checkbox list when user press a key.
    sort Function Nothing No Allow user to sort list by custom function.
    sortOptions Anything Undefined No Allow user to pass parameters in his custom sort.

    Methods

    Checked

    Return the list of ids of the checked items.

    $("myselector").dropdownCheckbox("checked");

    Unchecked

    Return the array of ids of the unchecked items.

    $("myselector").dropdownCheckbox("unchecked");

    Append

    Append an array of items.

    $("myselector").dropdownCheckbox("append", [{ ... }]);

    AppendOne

    Add one item.

    $("myselector").dropdownCheckbox("appendOne", { ... });

    Remove

    Remove multiple items from a array of ids

    $("myselector").dropdownCheckbox("append", [34, 56, 76]);

    Reset

    Reset the dropdown from an array of items.

    $("myselector").dropdownCheckbox("append", [{ ... }}]);

    FAQ

    What is an "item" ?

    An item contains three fields: id (integer), label (string), isChecked (boolean).
